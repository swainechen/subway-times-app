# subway-times-app

React web app to view the wait time for the next train. 

Run `npm start` to run. 

The data api that this app uses is in the repo `mta-api`: github.com/kalyani-subbiah/mta-api. It needs to be run (for example, on port 5000) and src/index.js needs to be updated with its URL and port to run this app.
